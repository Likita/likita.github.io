Canvas 2D API

  • Рисование линий, прямоугольников
  • Окружности, круги

Прямоугольник

strokeRect(x, y, width, height)
context.strokeRect(10, 10, 100, 100);


Закрашенный прямоугольник

fillRect(x, y, w, h)
context.fillRect(10, 10, 100, 100)

Очищение области

clearRect(x, y, width, height)

Настройки пера

      context.strokeStyle = '#000' // Цвет линии
      context.fillStyle = '#fff' // Цвет закраски
      context.lineWidth = 4  // Толщина линии

Курсор

      beginPath()   // новый контур
      moveTo(x,y) // курсор в (x, y)
      // линия от текущего положения к (x,y)
      lineTo(x,y)
      // соединения текущей точки и начальной
      closePath()

Окружность

      centerX, centerY, r, angle0, angle1, sunwise
      context.arc(55, 55, 50, 0, Math.PI*2, true);
      context.lineWidth = 4;
      context.stroke();
      context.fillStyle = "purple";
      context.fill();
