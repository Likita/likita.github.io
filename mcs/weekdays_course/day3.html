Свойство position Static

position: static;
  • Элемент без особого позиционирования
  • Стандартное поведение всех элементов по умолчанию

Свойство position Relative

position: relative;
  • Доступны свойства top, bottom, left, right
  • Элемент сдвигается относительно своего местоположения

Свойство position Absolute

position: absolute;
  • Доступны свойства top, bottom, left, right
  • Элемент располагается относительно ближайшего спозиционированного родителя
  • Элемент не занимает место в общем потоке
  • Ширина элемента по содержимому

Свойство position Fixed

position: fixed;
  • Доступны свойства top, bottom, left, right
  • Привязывается к координатам окна
  • При прокрутке элемент остаётся на том же месте

Собственные шрифты

  • Скачиваем нужный шрифт
  • Или находим на Google Fonts

Правило @font-face

Формат WOFF самый легковесный

      @font-face { 
         src: url(fonts/gentium.woff); 
         font-family: Gentium; 
         font-weight: 400; 
      }
